# 股票估值分析（宏观增强版）

本文档使用若干英文术语以保留金融学精确含义：**regime** = 宏观状态；**base case** = 基准情景；**Wave 1/2/3** = 三波研究循环；**FCF** = 自由现金流；**capex** = 资本开支；**ERP** = 股权风险溢价。

## 核心原则

1. 经营事实先行，估值随后。预测不是历史趋势的简单外推——外生变量显著影响公司价值时须启用宏观推演。
2. 先标准化盈利和现金流，再选倍数或折现框架。估值参数与当前 regime 匹配。
3. 抓住真正驱动价值的少数变量（通常 1–3 个），再展开情景与敏感性。
4. 五项判断决定估值锚：增长质量、现金流质量、资本效率、周期位置、风险溢价。
5. 任何结论须经反向校验——当前股价隐含了什么宏观路径、什么经营兑现、什么风险溢价。
6. 估值是迭代推演：信息分波次抓取（探索 → 聚焦 → 反证），核心变量可被新证据推翻；关键判断以显式推理链与反证条件呈现，不以叙述替代推演。

## 资料与搜索规范

### 可信度分层（从高到低）

1. 交易所公告、年报/中报/季报、业绩快报、10-K/10-Q/20-F、招股书、并购/定增/可转债材料
2. 业绩会实录、投资者交流材料、资本市场日材料
3. 行业协会、政府统计、海关、央行与监管、财政预算、官方利率与债券收益率、能源/运价/商品价格
4. 客户与供应商公告、同业财报与公告
5. 可验证的第三方数据库或财经终端
6. 券商研报仅作参照，不作为假设起点

### 信息分层

全程标注信息层级：
- **事实**——已披露、可核验
- **推断**——由事实延伸的判断
- **假设**——尚未被验证的预测输入

口径一致性是底线：报表口径、并表范围、股本基数、币种一旦混淆，后续数字失去意义。宏观/政策数据使用时标注时点，不把旧 regime 当作当前事实。

### 自适应搜索循环

每次执行信息获取（搜索、阅读文档、查询数据）须完成下列循环，不做无意图搜索。

**搜索前 · 声明意图**

- 本次要回答的**具体问题**（精确到假设层级，而非"了解公司"）
- **预期信号**：什么结果证实 / 什么结果证伪 / 什么是意外但值得追踪
- 在信息图谱中的**位置与优先级**

**搜索后 · 评估**

- **回答度**：完整 / 部分 / 未答 / 问题本身需修正
- **信息分层**：把所获内容归入事实 / 推断 / 假设
- **意外信号**：记录并判断是否立即追踪
- **图谱更新**：新证据并入图谱，调整活跃假设优先级

**下一步决策**（五选一，不可省略）

| 决策 | 触发条件 | 动作 |
|------|---------|------|
| 推进 | 当前问题已充分解答 | 进入图谱中下一优先级问题 |
| 深挖 | 方向正确但精度/覆盖不足 | 调整关键词、换数据源、寻找交叉验证 |
| 转向 | 发现更重要或更紧迫的新问题 | 加入图谱并提升优先级 |
| 回退 | 新信息推翻已锁定假设或核心变量 | 回到第 4 步更新变量集，记录触发证据与前后差异 |
| 标记缺口 | 连续搜索无增量或数据确不可得 | 停止方向，记录缺口对结论的影响 |

**空转保护**：同一问题连续 2 次搜索无增量，立即暂停诊断（意图是否模糊？关键词需调整？数据源已穷尽？问题需拆解？）。诊断后仍无进展，标记为信息缺口，不再空转。

### 信息图谱

动态维护的活地图，是每次搜索决策的依据：

| 层 | 内容 | 更新时机 |
|------|------|----------|
| 已确认事实 | 已验证可引用的经营与财务事实（含来源、时点） | 每次搜索后 |
| 活跃假设 | 当前正在验证或待验证的判断，按优先级排序 | 每次搜索后重排 |
| 信息缺口 | 尝试但未获得的关键信息，含对结论影响评估 | 标记缺口时 |
| 意外发现 | 预期外但可能重要的信号，待归类 | 搜索后发现时 |

### 研究波次（搜索的战略节奏）

波次规定各阶段的**搜索方向与停止条件**，由信息图谱判断是否达成。

| 波次 | 目标 | 停止条件 | 对应 Step |
|------|------|---------|----------|
| **Wave 1 · 建盘** | 广度事实底稿 | 能白描主业/财务节奏；列出 5–10 个候选关键变量 | Step 0–3 |
| **Wave 2 · 聚焦** | 验证/证伪候选变量，收敛至 1–3 个 | 每个候选变量能回答方向、量级、持续性、传导弹性、领先指标 | Step 4 |
| **Wave 3 · 反证** | 主动找推翻 base case 的证据 | 至少 2–3 条对 base case 有实质挑战的反证，每条完成量化评估 | Step 10 后、Step 11 前 |

**Wave 3 找不到任何实质反证** 本身是警示信号——要么估值确实鲁棒、要么搜索过窄陷入确认偏差，后者须立即补做一轮。

**动态回退**：Wave 2/3 发现新关键变量或证伪已锁定变量时，回到 Step 4 更新变量集，并在分析报告相应板块显式记录回退过程。隐藏回退会让估值退化为事后合理化。

**思维切换**：广度 / 聚焦 / 批判三种模式有意识区分——混着做往往三项都不到位。

### 推演卡片

关键判断以结构化卡片呈现，取代"因此""综合来看"等叙述掩盖推理跳跃的表达：

```
【结论】     一句话判断
【前提事实】 已验证事实（来源 / 时点 / 层级：事实|推断|假设）
【推理链】   A → B → C → 结论；每跳标注逻辑类型（量化 / 经验 / 类比 / 历史）
【置信度】   每跳标注 高/中/低，显式指出最脆弱一跳
【反证条件】 1–3 条可观测的失效信号
【信息缺口】 还不知道什么；能否补齐；补不齐的影响多大
```

**构建卡片的节点**（全流程通常 **5–7 张**，不堆砌）：
- 每个核心变量（Step 4）
- 关键估值参数（Step 8）
- 每个情景的假设组合（Step 10）

卡片是活的——新证据可调整置信度；反证可命中反证条件触发回退。

报告中不罗列卡片原文，但叙述须反映卡片内容：前提、推理链、最脆弱一跳、反证条件。

## 宏观增强模块

### 何时启用

满足任一条件即启用：

1. 收入/成本/现金流/估值显著受**利率、信用、汇率、商品、能源、运价、关税、补贴、监管、地缘事件**驱动
2. 行业属于**强周期、信用驱动、库存驱动、地产/基建/capex/政策驱动**，或估值高度依赖折现率与风险偏好
3. 显著**海外收入/采购/产能/跨币种负债**，或区域风险敞口集中
4. 历史数据跨越**制度切换**：加息转降息、疫情后修复、补贴退坡、出口限制、供应链迁移、战争/制裁、税制变化
5. 市场分歧的核心在外生变量方向或幅度，而非公司执行

均不满足时可不启用，但须在报告中说明理由。

### 工作方式（M1–M4）

**M1 · 定义当前 regime**——3–6 个最关键外生变量（超过 6 个通常说明主线未聚焦）。候选维度：

- 增长/需求：GDP、工业增加值、PMI、社零、地产/基建/制造业投资、企业 IT 预算、信用脉冲
- 通胀/成本：CPI、PPI、工资、原材料、能源、运价
- 利率/流动性：政策利率、国债收益率、信用利差、融资可得性
- 汇率/跨境：主要结算货币、资本流向、海外需求分化
- 政策/地缘：关税、制裁、补贴、监管强度、出口限制、冲突对供应链/需求的冲击

**M2 · 构建三路径**——至少 Bear / Base / Bull。每条写清：外生变量如何演变；设定依据（逻辑而非机械上下）；可持续验证或证伪的公开指标。

**M3 · 映射到公司层**——每个宏观变量落到具体传导端：

- 收入端：销量、客户预算、产能利用率、同店、渗透率、项目节奏、区域需求
- 价格/结构：ASP、ARPU、产品组合、折扣率、合同条款
- 成本：原材料、能源、人工、运费、采购币种
- 费用/capex：销售投入、研发节奏、扩产、维护性/扩张性 capex
- 现金流/资产负债：营运资本、存货、应收、再融资压力
- 估值：无风险利率、ERP、信用利差、可比倍数中枢

**M4 · 监控面板**——每个关键外生变量：当前观察值与方向；未来 3–12 个月领先指标；何种变化导致 base case 失效。

### 启用后对主流程的增补

所有宏观相关增补集中在本表，主流程不再分散 if/else。

| 主流程位置 | 启用宏观模块时的增补 |
|-----------|-------------------|
| Step 0 | 列出 3–6 个关键外生变量（M1） |
| Step 2 标准化 | 多拆一层：剔除低利率、汇率、原材料、补贴、税惠、景气位置的扰动，逼近中性环境盈利能力 |
| Step 3 传导 | 外生变量如何先作用于行业、再传导至公司；区分快变量（当季见效）与 1–4 季度时滞 |
| Step 4 核心变量 | 至少含 1 个外生变量，除非能论证影响微弱 |
| Step 5 预测 | 采用"外生变量 → 行业量价结构 → 公司量价结构 capex → 利润现金流"框架；每个假设标注来源 |
| Step 6 压力测试 | Bear 情景现金流/再融资/契约压力；汇率+商品+需求同时下修的弹性；能否撑过完整下行周期 |
| Step 7 方法选择 | 优先能清晰分离经营假设与估值参数的方法；成长股用 DCF/PE 交叉、周期用中周期利润/NAV/成本曲线、金融用 ROE/PB、多币种确保口径一致 |
| Step 8 参数 | 参数须与 base case 宏观环境一致 |
| Step 10 情景 | "宏观路径 × 公司执行"矩阵；敏感性加入 1–2 个关键外生变量（实际利率/商品/汇率/信用利差/运价） |
| Step 11 反向校验 | 额外倒推当前价格隐含的利率/汇率/商品/信用/政策路径，与 base case 对比 |
| Step 12 收敛 | 显式区分：哪些价值来自公司执行、哪些来自外部环境重估 |

## 分析主流程

每 Step 后括号标注对应波次。本流程为"宏观中性"叙述——宏观启用时的增补见上表。

### 第 0 步：界定估值对象、口径与宏观暴露（Wave 1）

- 合并报表范围覆盖的子公司/联营；少数股东权益规模
- 近 1–2 年并购、剥离、分拆、股权融资、回购注销、会计口径变更
- 历史数据可比性（不可比须先重述再做趋势判断）
- 主要暴露：区域、币种、融资、商品、政策
- 判断是否启用宏观模块

目标：不把并表变化误读为内生增长，不把资本运作混淆为经营改善，不把外部环境变化遗漏为公司好转。

### 第 1 步：历史经营事实底座（Wave 1）

**利润表**——近 3–5 年收入/毛利/营业利润/净利润；近 8 季度节奏；多业务/区域/币种分别拆解。

**资产负债表与现金流**——经营现金流、capex、FCF；存货、应收、合同负债、应付趋势；股本变化（回购、激励、可转债）。

**经营质量与外部弹性**——客户集中度、渠道结构、订单/产能；收入与现金流匹配度；归因——多少来自执行、多少来自利率/汇率/商品/周期/政策。

回答四问（不急下结论）：增长从哪来？利润率变化由什么驱动？利润与现金流质量是否一致？哪些是可持续能力、哪些是外部顺风？

### 第 2 步：标准化盈利与现金流（Wave 1 收尾）

**盈利端**剥离：

- 资产处置与投资收益
- 公允价值变动
- 汇兑损益
- 政府补贴
- 并购整合费用
- 一次性税收优惠

注意 non-GAAP 调整项中哪些确属一次性、哪些年年发生。

**现金流端**审视：

- 经营现金流是否被拉长账期/增加预收/保理美化
- capex 中维护性 vs 扩张性比例
- 租赁负债与资本化研发是否掩盖真实消耗
- 利润增长是否越来越依赖营运资本与 capex

完成后回答三问：真实盈利能力区间？现金创造相对利润是偏强还是偏弱？利润率变化是结构性、周期性、政策性还是一次性？

### 第 3 步：商业模式、行业结构、管理层（Wave 1 收尾）

**商业模式与行业**：

- 公司卖什么、客户为何买、替代品
- 价值链段位：品牌 / 制造 / 平台 / 资源
- 增长驱动：放量 / 提价 / 结构改善 / 景气
- 竞争优势根基：成本 / 技术 / 品牌 / 网络 / 政策
- 行业状态：稳态 / 份额迁移 / 强周期

**管理层与治理**：

- 指引兑现率
- 历次重大资本配置（投资/并购/分红/回购）事后回报
- 大小股东利益对齐（关联交易、体外资产、资金占用）
- 增减持与公开表态一致性
- 股权激励考核条件

### 第 4 步：锁定核心变量并定向验证（Wave 2）

**锁定 1–3 个核心变量**——真正主导未来一两年估值的变量通常不超过三个，不可所有指标等权。选定后阐明：为什么选这几个；哪些可控 / 哪些外生；领先指标；市场当前隐含的预期。

**定向验证**——不得直接进入预测，须先执行 Wave 2。对每个核心变量：

1. **子问题**——方向、量级、持续性、传导弹性、历史可比情境
2. **去哪里找什么**——业绩会问答、管理层量化指引、细分披露、同业最新季报、行业/政府/海关数据、政策文件、商品/运价/汇率/利率、客户供应商互证。抓取前明确预期信号与证伪信号
3. **领先指标**——未来 3–12 个月前置信号
4. **抓取后动作**——证据强化 / 削弱 / 推翻假设；推翻触发变量集回退；削弱调整对应推演卡片的置信度

**示例**：

- "AI capex 弹性"→ 超大规模厂商 capex 指引与履约、ODM 订单与产能、光模块/HBM 出货节奏、电力/机房瓶颈；领先指标：头部云厂 capex YoY、ODM 出货 MoM、关键器件交付周期
- "某车企单车盈利"→ 车型配置定价、终端折扣、电池/芯片/稀土成本曲线、产能利用率、同级竞品毛利；领先指标：周度终端销量、渠道库存、碳酸锂价格

**产出**：每个核心变量对应一张推演卡片（格式见"推演卡片"）。

变量选择不是终局——Wave 2/3 新证据触发回退时，更新变量集并在报告中记录。

### 第 5 步：分业务、分驱动预测

- 收入落到可解释经营驱动，不直接拍增速；不简单前推历史 CAGR
- 大客户或项目制公司按客户/产品线拆；披露有限时做量/价/结构/并表/汇率五因子拆解
- 利润率不可当自由参数：
  - 毛利率变化桥：价格、成本、利用率、结构、汇率、商品成本
  - 费用率：规模效应、扩张投入、激励摊销、区域扩张
  - 税率与少数股东损益对 EPS 的扰动
- 同步处理营运资本、capex 节奏、再融资、完全摊薄股本，口径与估值一致
- 每个关键假设标注来源：历史惯性 / 公司执行 / 宏观情景 / 政策地缘

### 第 6 步：资本效率、现金流质量与资产负债表韧性

预测后的质量检查：

- 收入增长对应的营运资本变化是否合理
- 利润率假设是否建立在过于乐观的利用率或成本上
- FCF 方向与利润增长是否一致
- capex 与研发是否短期大量消耗现金
- ROIC 与增量 ROIC 水平

若利润亮眼但现金流恶化、资本效率下滑、融资弹性不足，估值打折扣或回头重审增长可持续性。

### 第 7 步：选择估值方法

方法跟商业模式走：

- **PE**——盈利稳定、利润与现金流匹配
- **EV/EBIT 或 EV/EBITDA**——折旧重、资本结构差异大
- **PB/ROE**——资产负债表驱动（银行、保险）
- **DCF**——现金流可预测、可长期外推
- **SOTP**——多业务差异大或有分拆预期
- **周期股**——中周期利润，不在景气顶点硬外推
- **未盈利 biotech**——风险调整估值或事件树

选定后确认：估值口径与利润口径匹配；估值年份对准经营拐点；主方法与交叉验证基于不同假设；无同一逻辑重复计入。

### 第 8 步：估值参数的四维判断

不套用历史均值或行业均值。四维度判断，最终参数以推演卡片呈现：

- **质量**——增长来自量/价/结构哪个更可持续；利润率改善依赖经营质量还是一次性；现金流与资产负债表健康度
- **周期**——当前利润处于低谷/中枢/高点；供给改善 vs 需求透支；库存/价格/产能周期有无错配
- **风险溢价/折现率**——无风险利率、信用利差、国家/区域风险、融资环境、股权风险偏好
- **隐含预期**——股价已定价了怎样的增速、利润率、风险溢价；base case 是真正高于市场预期，还是沿用旧环境估值锚

参照系：公司自身历史估值区间、同业分位数、同业在增长/利润率/现金流/商业模式/宏观暴露上的差异。

**硬约束**：

- 不在景气高点利润上叠加高倍数
- 不给低质量增长配高质量估值
- 不将同一增长逻辑同时计入盈利预测与估值溢价
- 不把零利率或单一政策红利阶段的估值中枢搬至当前 regime

### 第 9 步：EV 到股权价值桥接

完整桥接：

- 扣净债务
- 处理少数股东权益、可转债、租赁负债等类债项目
- 单列非核心资产与联营公司权益
- 以保守的完全摊薄股本计算每股价值

SOTP 时检查：

- 总部费用是否重复扣减
- 现金债务是否重复计算
- 税收与交叉持股是否遗漏

跨币种债务、海外资产/现金、跨境分红限制、再融资压力——明确币种与融资成本口径。

**资本回报策略评估**（决定利润增长能否转化为每股价值增长）：

- 分红 payout 与趋势
- 回购规模节奏（真注销 vs 对冲激励）
- 管理层对自由现金流的优先选择（分红/回购/再投/并购/留存）
- 近几年完全摊薄股本是收缩还是膨胀

### 第 10 步：三情景与敏感性

围绕核心变量构建 Bear / Base / Bull，每个情景对应一张独立推演卡片。三张卡片间的差异须显性化——哪些假设变了、为什么变、变量间相关性（如需求下行往往伴随定价压力与汇率承压）。

- **Bear**——底部在哪里
- **Base**——需要怎样的环境与经营兑现支撑中枢
- **Bull**——上行来自执行超预期、风险溢价回落、还是外部改善，分开评估

敏感性优先测试：销量/用户增长、ASP/结构、毛利率、capex/FCF 转换率、估值倍数/折现率。目标不是堆表格，而是找"最脆弱的那个假设"。

可给概率加权目标价，须说明概率是工作假设，不得伪精确到小数点。

### 第 11 步：反向校验与反证处理（Wave 3 成果）

**反证处理**——对 Wave 3 清单中每条证据做权重评估（接受 / 修正 / 拒绝）与依据。

**反向预期校验**——倒推当前股价隐含的收入增速、利润率、ROIC、风险溢价。若 base case 仅略高于市场已定价预期，上行判断须克制。

**一致预期比较**——逐项对比收入、毛利率、净利润、EPS 与市场一致预期。偏差 >5% 须说明分歧依据；与一致几乎一致时坦诚指出——说明本次分析可能无增量信息。

### 第 12 步：收敛为可解释、可跟踪、可更新的判断

收敛至四层面：

- **经营**——未来一两年最关键的兑现节点
- **宏观**——哪些外生变量决定预测成立（若启用）
- **估值**——当前价与合理价值的差距来自业绩差、预期差还是风险溢价差
- **证伪**——哪几个可跟踪数据点最可能推翻判断

最终确认四点：

1. 结论由事实与逻辑推导，非预设立场倒推
2. 关键假设数量足够少，每项可靠公开数据跟踪
3. 未被当前股价、市场情绪、卖方一致预期带偏
4. 宏观启用时，已区分公司执行贡献与外部环境重估贡献

## 行业适配

每类在核心指标基础上给出宏观补充维度。

### 制造 / 汽车零部件 / 工业
核心：出货量、ASP、单车价值量、产能利用率、capex、客户集中度。项目制公司爬坡进度比收入增速更重要。宏观补充：制造业投资、下游 capex、汇率、关税、原材料、运费。

### 消费
核心：同店增长、渠道扩张、终端折扣、库存健康、提价能力、复购率。关键判断：增长来自真实需求，还是渠道补库存 + 提价的表观繁荣。宏观补充：居民收入、就业、消费信心、地产财富效应、食品与原材料成本。

### 互联网 / 软件 / 平台
核心：用户增长与留存、付费率、ARPU、获客成本、贡献利润率、股票薪酬。经营杠杆是估值弹性主要来源。宏观补充：广告预算、企业 IT 支出、融资环境、实际利率对成长股估值影响。

### 半导体 / 硬件
核心：ASP、良率、产能利用率、渠道库存、设计赢单管线、技术代际。周期位置判断不可省略——决定应以什么利润水平乘倍数。宏观补充：全球电子需求、服务器/AI capex、出口限制、汇率、设备投资周期。

### 银行 / 保险 / 券商
资产负债表逻辑：ROE、资产质量、拨备、资本充足率、负债成本、投资收益可持续性。不可套用 EV 口径。宏观补充：利率曲线、信用成本、地产与就业、监管导向、资本市场活跃度。

### 周期 / 资源 / 化工 / 航运
核心：价格周期位置、库存水位、供给约束、成本曲线、新产能投放。以中周期利润或资产价值视角——景气高点利润直接外推是最常见错误。宏观补充：中国/全球需求、能源、美元、航运路线、制裁与贸易政策。

### 未盈利 biotech / 早期硬科技
核心：里程碑兑现概率、现金消耗、下一轮融资需求、商业化路径。产出是价值区间与事件敏感性，非精确点估值。宏观补充：融资环境、利率、审批节奏、支付方政策、关键设备/技术出口限制。

## 输出格式

两部分：**简要结论在前，分析报告在后**。

### 简要结论

至少包含：股票名称、代码、当前股价；是否启用宏观模块；12 个月基准目标价及乐观/悲观区间；可选概率加权目标价；基准情景核心盈利预测（营收、净利润、EPS/FCF）与估值方法/参数；相对股价空间；宏观路径一句话（若启用）；估值逻辑一两句；最大上行催化与最大下行风险各一条；2–3 个关键跟踪指标；免责声明。

**示例**（数字为占位符）：

```
# [股票名称]（[代码]）估值结论

> 免责声明：本分析不构成投资建议，仅为基于公开信息的独立估值练习。市场有风险，投资需谨慎。

| 项目 | 数值 |
|------|------|
| 当前股价 | ¥XX.XX |
| 宏观增强模块 | 启用 / 未启用 |
| **12 个月目标价（基准）** | **¥XX.XX** |
| 概率加权目标价（可选） | ¥XX.XX |
| 乐观情景 | ¥XX.XX |
| 悲观情景 | ¥XX.XX |
| 基准上行空间 | +XX% |
| 核心假设 | FY20XXE 营收 XXX 亿元，净利润 XXX 亿元，XX 倍 PE / XX% WACC |
| 宏观基准路径 | [实际利率回落、库存去化结束、美元走弱] |

**估值逻辑：** [1–2 句话]
**最大上行催化：** [一句话]
**最大下行风险：** [一句话]
**关键跟踪指标：** [2–3 个]
```

### 分析报告

以叙述段落组织，按板块推进：

1. 公司概况与口径说明
2. 经营事实与历史归因
3. 标准化盈利
4. 商业模式与竞争格局（含宏观传导，若启用）
5. 核心变量与预测
6. 宏观环境与情景路径（仅启用时）
7. 估值
8. 情景与敏感性
9. 反向校验与风险
10. 结论与跟踪框架

**叙述原则**：关键判断的前提、推理链与反证条件在叙述中自然呈现——读者应能看到"从 X 如何推到 Y、哪一跳最脆弱、什么会让它失效"。Wave 2/3 触发过的回退显式记录。板块详略按公司情况调整。

## 防错清单

1. 订单额 / 定点金额 / 意向协议不等于收入；一次性收益不等于长期盈利能力；管理层指引是假设来源，不是事实。
2. 始终使用完全摊薄股本，不可忽略未来稀释。
3. 行业叙事不能替代收入、利润率、现金流、股本这四个硬变量。
4. 经营模型未完成之前不锚定估值结论；当前股价仅用于反向校验，不作估值输入。
5. 景气高点利润上不叠加高倍数；不同商业模式不机械套用同业平均倍数。
6. regime change 出现时，历史均值与历史倍数参考权重主动下调；不搬零利率或单一政策红利阶段的估值中枢至当前 regime。
7. 地缘政治落到具体传导路径（需求 / 供给 / 成本 / 汇率 / 风险溢价），不作空泛风险条目。
8. 概率加权目标价中的概率是工作假设，不得伪精确到小数点。
