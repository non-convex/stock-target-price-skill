# 股票估值分析（宏观增强版）

## 核心原则

1. 经营事实先行，估值随后。预测不是历史趋势的简单外推——外生变量显著影响公司价值时，须启用宏观推演。
2. 先标准化盈利和现金流，再选倍数或折现框架。估值参数须与当前宏观 regime 匹配。
3. 抓住真正驱动价值的少数变量，再展开情景和敏感性。变量过多通常说明主线未理清。
4. 增长质量、现金流质量、资本效率、周期位置、风险溢价——五项判断到位，估值锚才稳。
5. 任何结论都须经受反向校验：当前股价隐含了什么宏观路径、什么经营兑现、什么风险溢价。
6. 宏观分析须落到量、价、结构、利润率、资本开支、现金流与估值倍数上，不能只停留在叙事。
7. 估值是迭代推演。信息分波次抓取（探索 → 聚焦 → 反证），核心变量可被新证据推翻，关键判断须以显式推理链与反证条件呈现。

---

## 资料使用原则

**可信度从高到低**：

1. 交易所公告、年报/中报/季报、业绩快报、10-K/10-Q/20-F、招股书、并购/定增/可转债材料。
2. 业绩会实录、投资者交流材料、资本市场日材料。
3. 行业协会、政府统计、海关、央行与监管机构、财政预算、官方利率与债券收益率、能源/运价/商品价格等可验证数据。
4. 客户与供应商公告、同业财报与公告。
5. 可验证的第三方数据库或财经终端。
6. 券商研报仅作参照，不作为假设起点。

**信息分层**：事实（已披露、可核验）→ 推断（由事实延伸的判断）→ 假设（尚未被验证的预测输入）。全程标注。

**口径一致性是底线**：报表口径、并表范围、股本基数、币种一旦混淆，后续数字失去意义。

**数据时点**：宏观/政策数据使用时须标注时点，不得把旧 regime 当作当前事实。

## 工作方法论

本节规定"如何抓信息"与"如何把信息变成判断"。两者贯穿整个分析主流程。

### 研究波次（Research Waves）

信息抓取分三波循环，每一波目标与停止条件明确，后一波搜索方向由前一波发现决定。

| 波次 | 目标 | 主要抓取对象 | 停止条件 | 输出物 |
|------|------|-------------|---------|--------|
| **Wave 1 · 建盘**<br>（广度优先） | 回答"这家公司长什么样" | 年报、季报、业绩会、官网业务介绍、监管披露、同业识别 | 能白描主业/客户/财务节奏；初步列出 5–10 个候选关键变量 | 事实底稿 + 候选变量清单 |
| **Wave 2 · 聚焦**<br>（围绕候选变量深挖） | 验证或证伪候选变量，收敛至 1–3 个真正驱动变量 | 业绩会问答、管理层量化指引、细分 ASP/销量/利用率、客户与供应商互证、行业/政府/海关数据、同业最新披露、领先指标 | 每个候选变量都能回答：方向、量级、持续性、传导弹性、领先指标 | 每个核心变量的证据包 + 推演卡片 |
| **Wave 3 · 反证**<br>（主动找推翻证据） | 对冲确认偏差 | 空头观点、竞品动作与指引、监管反向信号、历史失败案例、客户流失、供应链风险、管理层指引落空记录、减持动向 | 至少 2–3 条对 base case 有实质挑战的反证，每条完成量化评估 | 反证清单 + 每条的处理方式 |

**动态回退**：Wave 2/3 发现新关键变量、或证伪已锁定变量时，须显式回到第 4 步更新变量集，并在分析报告中记录触发证据与前后差异。隐藏回退过程会让估值退化为事后合理化。

**思维切换**：广度 / 聚焦 / 批判三种模式须有意识区分，混着做往往三项都不到位。

### 推演卡片

关键判断须组织为显式推理结构，而非叙述。卡片是工作载体，报告中自然呈现前提、推理链与反证条件即可。

```
【结论】     一句话写清判断
【前提事实】 已验证事实（标注：来源 / 时点 / 层级：事实|推断|假设）
【推理链】   A → B → C → 结论；每跳标注逻辑类型（量化 / 经验 / 类比 / 历史）
【置信度】   每跳标注 高/中/低，显式指出最脆弱那一跳
【反证条件】 1–3 条可观测的失效信号
【信息缺口】 还不知道什么；能否在 Wave 2/3 补齐；补不齐的影响多大
```

**必须构建卡片的节点**：第 4 步核心变量；第 5 步每项增长驱动、利润率桥、资本开支假设；第 8 步估值参数；第 10 步每个情景的假设组合;第 11 步对隐含预期与一致预期差异的解释。

**卡片是活的**：Wave 2 证据可调整置信度，Wave 3 反证可命中反证条件触发回退。

## 宏观增强模块

### 何时启用

满足以下任一条件须启用，不得仅做 bottom-up 外推：

1. 公司收入/成本/现金流/估值显著受**利率、信用、汇率、商品价格、能源、运价、关税、补贴、监管、地缘事件**驱动。
2. 行业属于**强周期、信用驱动、库存驱动、地产/基建驱动、资本开支驱动或政策驱动**，或估值高度依赖折现率与风险偏好。
3. 公司有显著**海外收入、海外采购、海外产能、跨币种负债**，或区域风险敞口集中。
4. 历史数据跨越了**制度切换**：加息转降息、疫情后修复、补贴退坡、出口限制、供应链迁移、战争/制裁、税制变化。
5. 市场分歧的核心不在公司执行，而在外生变量的方向和幅度。

均不满足时可不启用，但须在报告中明确说明不启用的理由。

### 工作方式（M1–M4）

**M1 · 定义当前 regime**——选取 3–6 个最关键外生变量（超过 6 个通常说明主线未聚焦）。候选维度：
- 增长/需求：GDP、工业增加值、PMI、社零、地产/基建/制造业投资、企业 IT 预算、信用脉冲
- 通胀/成本：CPI、PPI、工资、原材料、能源、运价
- 利率/流动性：政策利率、国债收益率、信用利差、融资可得性
- 汇率/跨境：主要结算货币、资本流向、海外需求分化
- 政策/地缘：关税、制裁、补贴、监管强度、出口限制、冲突对供应链/需求的冲击

**M2 · 构建宏观路径**——至少 Bear / Base / Bull 三条。每条写清：外生变量如何演变；设定依据（逻辑而非机械上下调整）；可持续验证或证伪的公开指标。路径向公司层的映射留给主流程第 5 和第 10 步。

**M3 · 映射到公司层**——每个宏观变量落到具体传导端，不停留在叙事：
- 收入端：销量、客户预算、产能利用率、同店、渗透率、项目节奏、区域需求
- 价格/结构端：ASP、ARPU、产品组合、折扣率、合同价格条款
- 成本端：原材料、能源、人工、运费、采购币种
- 费用/capex 端：销售投入、研发节奏、扩产、维护性/扩张性 capex
- 现金流/资产负债端:营运资本、存货风险、应收、再融资压力
- 估值端：无风险利率、股权风险溢价、信用利差、可比倍数中枢

**M4 · 监控面板**——对每个关键外生变量给出：当前观察值与方向；未来 3–12 个月可跟踪的领先指标；何种变化会导致 base case 失效。

## 分析主流程

### 第 0 步：界定估值对象、口径与宏观暴露

上市主体合并范围覆盖哪些子公司/联营；少数股东权益规模；近一两年有无并购、剥离、分拆、股权融资、回购注销或口径变更；历史数据是否可比——不可比须先重述再做趋势判断。

同步判断：公司主要的**区域、币种、融资、商品、政策**暴露；是否触发宏观模块；若触发，列出 3–6 个关键外生变量。

目标：不把并表变化误读为内生增长，不把资本运作混淆为经营改善，不把外部环境变化遗漏为公司好转。

### 第 1 步：历史经营事实底座（Wave 1 的主要成果）

从三个维度还原经营面貌。

**利润表**——近 3–5 年收入/毛利/营业利润/净利润，近 8 季度节奏；多业务/多区域/多币种须分别拆解。

**资产负债表与现金流**——经营现金流、capex、FCF 为基本功；关注存货、应收、合同负债、应付的趋势；股本是否悄然膨胀（回购、激励、可转债）。

**经营质量与外部弹性**——客户集中度、渠道结构、订单/产能等领先指标；收入增长与现金流是否匹配；归因：历史变化中多少来自执行、多少来自利率/汇率/商品/周期/政策。

先回答四问，不急下结论：增长从哪来？利润率变化由什么驱动？利润与现金流质量是否一致？哪些是可持续能力、哪些是外部顺风？

### 第 2 步：标准化盈利与现金流

**盈利端**——剥离资产处置、投资收益、公允价值变动、汇兑、政府补贴、并购整合费用、一次性税惠；注意 non-GAAP 调整项中哪些确属一次性、哪些年年发生。

**现金流端**——经营现金流是否被拉长账期/增加预收/保理等美化；capex 中维护性 vs 扩张性比例；租赁负债和资本化研发是否掩盖真实资金消耗；利润增长是否越来越依赖营运资本和 capex。

宏观模块启用时多拆一层：过去盈利/现金流中受**低利率、汇率、原材料、补贴、税惠、景气位置**扰动的部分。标准化不仅剔除一次性项，还要逼近**中性环境下的盈利能力**。

完成后回答三问：真实盈利能力区间？现金创造 vs 利润是偏强还是偏弱？利润率变化是结构性、周期性、政策性还是一次性？

### 第 3 步：商业模式、行业结构、管理层与宏观传导

**商业模式与行业**——估值差异源于产业链位置，不只是增速：公司卖什么、客户为何买、替代品是什么；价值链段位（品牌/制造/平台/资源）；增长依赖放量、提价、结构改善还是景气；竞争优势根基（成本/技术/品牌/网络/政策）；行业是稳态、份额迁移还是强周期。

**管理层与治理**——同样利润水平，治理优劣给出的倍数天差地别：指引兑现率；历次重大资本配置（投资/并购/分红/回购）的事后回报；大股东与中小股东利益是否对齐（关联交易、体外资产、资金占用）；增减持与公开表态一致性；股权激励考核条件。

**宏观传导**（宏观模块启用时）——外生变量如何先作用于行业、再传导至公司？哪些影响需求端、成本端、估值倍数/折现率？哪些是快变量（当季见效）、哪些有 1–4 季度时滞？

### 第 4 步：锁定 1–3 个核心变量

真正主导未来一两年估值变化的变量通常不超过三个，不可所有指标等权罗列。

选定后阐明四点：为什么选这几个而非其他；哪些可控、哪些外生；有哪些领先指标；市场当前隐含了什么预期。

宏观模块启用时，核心变量中通常至少包含 **1 个外生变量**，除非能论证外生变量影响确实微弱。

**本步选择不是终局**——Wave 2/3 新证据可触发回退，详见"工作方法论 · 动态回退"。

### 第 4.5 步：围绕核心变量的定向深抓（Wave 2 的主要载体）

锁定变量后不得直接进入第 5 步，须先执行 Wave 2。对每个核心变量列出四项：

1. **需要验证的子问题**——方向、量级、持续性、传导弹性、历史可比情境。
2. **去哪里找什么**——业绩会问答、量化指引、细分披露、同业最新季报、行业/政府/海关数据、政策文件、商品/运价/汇率/利率、客户与供应商互证。抓取前明确"预期看到什么数据、什么数据会改变判断"。
3. **领先指标**——未来 3–12 个月前置信号，用于 base case 持续验证。
4. **抓取后动作**——证据强化、削弱还是推翻假设？推翻触发第 4 步回退；削弱调整推演卡片对应跳的置信度。

**示例**：
- 变量"AI capex 弹性"→ 抓取：超大规模厂商 capex 指引与履约、ODM 订单与产能、光模块/HBM 出货节奏、电力/机房瓶颈；领先指标：头部云厂 capex YoY、ODM 出货 MoM、关键器件交付周期。
- 变量"某车企单车盈利"→ 抓取：车型配置定价、终端折扣、电池/芯片/稀土成本曲线、产能利用率、同级竞品毛利；领先指标：周度终端销量、渠道库存、碳酸锂价格。

每个核心变量完成后对应一张推演卡片，在报告"核心变量与预测"板块以叙述方式呈现。

### 第 5 步：分业务、分驱动、分情景的预测

收入预测须落到可解释的经营驱动，不能直接拍增速；宏观增强模式下不能把过去 3 年 CAGR 简单前推。

优先采用"**外生变量 → 行业量价结构 → 公司量/价/结构/capex → 利润与现金流**"框架。大客户或项目制公司按客户/产品线拆分；披露有限时做量/价/结构/并表/汇率五因子拆解，标注哪些因子由宏观情景驱动。

**利润率不可当自由参数**，须构建变化桥：毛利率（价格、成本、利用率、结构、汇率、商品成本）；费用率（规模效应、扩张投入、激励摊销、区域扩张）；税率与少数股东损益对 EPS 的扰动。

同时处理营运资本、capex 节奏、再融资压力、完全摊薄股本，确保预测口径与估值口径一致。

每个关键假设标注来源：**历史惯性 / 公司执行改善或恶化 / 宏观情景变化 / 政策地缘影响**，并对应推演卡片（详见"工作方法论"）。

### 第 6 步：资本效率、现金流质量与资产负债表韧性

预测后先做一轮质量检查：收入增长对应的营运资本变化是否合理？利润率假设是否建立在过于乐观的利用率或成本前提？FCF 方向与利润增长是否一致？capex 和研发是否短期大量消耗现金？ROIC 和增量 ROIC 水平如何？

宏观模块启用时补充三项压力测试：

1. Bear 情景下现金流是否转负、再融资成本是否上升、债务契约是否承压。
2. 汇率、商品、需求同时下修时的利润率和现金流弹性。
3. 资产负债表能否撑过完整下行周期，而非仅一两个季度。

若利润亮眼但现金流恶化、资本效率下滑、融资弹性不足，估值须打折扣或回头重审增长可持续性。

### 第 7 步：选择估值方法

方法跟着商业模式走：

- **PE**：盈利稳定、利润与现金流匹配
- **EV/EBIT 或 EV/EBITDA**：折旧重、资本结构差异大
- **PB/ROE**：资产负债表驱动（银行、保险）
- **DCF**：现金流可预测、可长期外推
- **SOTP**：多业务差异大或有分拆预期的控股平台
- **周期股**：用中周期利润估值，不在景气顶点硬外推
- **未盈利 biotech**：风险调整估值或事件树

宏观模块启用时优先选择**能清晰分离经营假设与估值参数**的方法，必要时主方法 + 交叉验证：

- 利率/风险溢价敏感的成长股：DCF 或 PE/EV 交叉，折现率与退出倍数须与宏观情景对应
- 周期/资源股：中周期利润、NAV、成本曲线，避免现货高价与高倍数叠加
- 金融股：ROE/PB 结合信用成本、资本充足率、负债成本
- 多区域/多币种：估值口径与汇率口径一致

选定后确认四点：估值口径与利润口径是否匹配；估值年份是否对准经营拐点；主方法与交叉验证是否基于不同假设；有无同一宏观利好重复计入盈利和倍数。

### 第 8 步：估值参数的四维判断

不可直接套用历史均值或行业均值。最终参数选择以推演卡片呈现。从四个维度判断：

- **质量**——增长来自量/价/结构哪个更可持续；利润率改善依赖经营质量还是一次性因素；现金流与资产负债表健康度
- **周期**——当前利润处于低谷/中枢/高点；供给改善 vs 需求透支；库存/价格/产能周期有无错配
- **风险溢价/折现率**——无风险利率、信用利差、国家/区域风险、融资环境、股权风险偏好；参数是否与 base case 宏观环境一致
- **隐含预期**——股价已定价了怎样的增速、利润率和风险溢价；base case 是真正高于市场预期，还是只是沿用旧环境估值锚

参数参照系：公司自身历史估值区间、同业分位数、同业在增长/利润率/现金流/商业模式/宏观暴露上的差异。

**四条硬约束**：

1. 不在景气高点利润上叠加高倍数。
2. 不给低质量增长配高质量公司估值。
3. 不将同一增长逻辑同时计入盈利预测和估值溢价。
4. 不将零利率时代或单一政策红利阶段的估值中枢机械搬至当前 regime。

### 第 9 步：企业价值到股权价值的桥接

EV 或利润倍数不够，须完成桥接：扣净债务；处理少数股东权益、可转债、租赁负债等类债项目；单列非核心资产与联营公司权益；以保守的完全摊薄股本计算每股价值。SOTP 时留意总部费用是否重复扣减、现金债务是否重复计算、税收与交叉持股是否遗漏。

跨币种债务、海外资产/现金、跨境分红限制、再融资压力须明确币种和融资成本口径。

评估资本回报策略（决定利润增长能否转化为每股价值增长）：分红 payout 与趋势；回购规模节奏（真注销 vs 对冲激励）；管理层对自由现金流的优先选择（分红/回购/再投/并购/留存）；近几年完全摊薄股本是收缩还是膨胀。

### 第 10 步：三情景与敏感性

围绕第 4 步核心变量构建三情景，每个情景对应一张独立推演卡片。三张卡片间的差异须显性化：哪些假设变了、为什么变、变量间相关性（如需求下行往往伴随定价压力与汇率承压）。

宏观模块未启用时按经营变量做 Bear / Base / Bull。启用时做"**宏观路径 × 公司执行**"矩阵：

- **Bear**：外部恶化 + 执行不达预期，底部在哪
- **Base**：需要怎样的宏观 + 经营兑现才能支撑中枢
- **Bull**：上行来自外部改善、执行超预期、还是风险溢价回落——三者分开评估

敏感性优先测试：销量/用户增长、ASP/结构、毛利率、capex 与现金流转换率、估值倍数/折现率；宏观模块启用时再加 1–2 个关键外生变量（实际利率、商品价格、汇率、信用利差、运价）。目标不是堆表格，而是找"最脆弱的那个假设"。

必要时可给**概率加权目标价**，须说明概率是工作假设，不应伪精确到小数点。

### 第 11 步：反向校验、市场定价拆解、反证处理

本步直接对接 Wave 3 反证清单——每条反证在此处完成权重评估（接受 / 修正 / 拒绝）与依据，不可回避。若 Wave 3 未找到任何实质反证，本身是警示信号：要么估值确实鲁棒、要么搜索过窄——后者须立即补做一轮。

**反向预期校验**——倒推当前股价隐含的收入增速、利润率、ROIC、风险溢价。若 base case 仅略高于市场已定价预期，上行空间判断须克制。

**宏观隐含路径校验**（若启用）——当前价格隐含了怎样的利率、汇率、商品、信用、政策环境？比 base case 更乐观还是更悲观？

**一致预期比较**——逐项对比关键预测（收入、毛利率、净利润、EPS）与市场一致预期。偏差 >5% 须说明分歧依据；若几乎一致，也要坦诚指出——说明本次分析可能无增量信息。

### 第 12 步：收敛为可解释、可跟踪、可更新的判断

收敛至四个层面：

- **经营**——未来一两年最关键的兑现节点
- **宏观**——哪些外生变量决定预测成立
- **估值**——当前价与合理价值的差距主要来自业绩差、预期差还是风险溢价差
- **证伪**——哪几个可跟踪数据点最可能推翻判断

最后确认四点：

1. 结论由事实和逻辑推导，而非预设立场倒推。
2. 关键假设数量足够少，每一项可靠公开数据持续跟踪。
3. 结论未被当前股价、市场情绪、卖方一致预期带偏。
4. 宏观模块启用时，已区分哪些价值来自公司执行、哪些来自外部环境引发的重估。

## 行业适配

### 制造业 / 汽车零部件 / 工业
核心关注：出货量、ASP、单车价值量、产能利用率、capex、客户集中度。有项目 SOP 节奏的公司，爬坡进度比收入增速更重要。宏观补充：制造业投资、下游 capex、汇率、关税、原材料、运费。

### 消费
核心关注：同店增长、渠道扩张、终端折扣、库存健康度、提价能力、复购率。关键判断：增长来自真实需求，还是渠道补库存 + 提价的表观繁荣。宏观补充：居民收入、就业、消费信心、地产财富效应、食品与原材料成本。

### 互联网 / 软件 / 平台
核心关注：用户增长与留存、付费率、ARPU、获客成本、贡献利润率、股票薪酬。经营杠杆是估值弹性主要来源。宏观补充：广告预算、企业 IT 支出、融资环境、实际利率对成长股估值的影响。

### 半导体 / 硬件
核心关注：ASP、良率、产能利用率、渠道库存、设计赢单管线、技术代际。周期位置判断不可省略——当前利润处于周期中枢、上沿还是下沿，决定应以什么利润水平乘倍数。宏观补充：全球电子需求、服务器/AI capex、出口限制、汇率、设备投资周期。

### 银行 / 保险 / 券商
以资产负债表逻辑为主：ROE、资产质量、拨备、资本充足率、负债成本、投资收益可持续性。不可套用 EV 口径。宏观补充：利率曲线、信用成本、地产与就业、监管导向、资本市场活跃度。

### 周期 / 资源 / 化工 / 航运
核心关注：价格周期位置、库存水位、供给约束、成本曲线、新产能投放。以中周期利润或资产价值视角审视——景气高点利润直接外推是最常见错误。宏观补充：中国/全球需求、能源、美元、航运路线、制裁与贸易政策。

### 未盈利 biotech / 早期硬科技
核心关注：里程碑兑现概率、现金消耗、下一轮融资需求、商业化路径。产出的是价值区间与事件敏感性，非精确点估值。宏观补充：融资环境、利率、审批节奏、支付方政策、关键设备/技术出口限制。

## 输出格式

最终输出两部分：**简要结论在前，分析报告在后**。读者可先看结论获取核心判断，再按需阅读报告。

### 第一部分：简要结论

至少包含：股票名称、代码、当前股价；是否启用宏观模块；12 个月基准目标价及乐观/悲观区间；可选的概率加权目标价；基准情景核心盈利预测（营收、净利润、EPS/FCF）与估值方法/参数；相对股价空间；宏观路径一句话（若启用）；估值逻辑一两句话；最大上行催化与最大下行风险各一条；2–3 个关键跟踪指标；免责声明。

**格式示例**（数字为占位符）：

```
# [股票名称]（[代码]）估值结论

> 免责声明：本分析不构成投资建议，仅为基于公开信息的独立估值练习。市场有风险，投资需谨慎。

| 项目 | 数值 |
|------|------|
| 当前股价 | ¥XX.XX |
| 宏观增强模块 | 启用 / 未启用 |
| **12 个月目标价（基准）** | **¥XX.XX** |
| 概率加权目标价（可选） | ¥XX.XX |
| 乐观情景 | ¥XX.XX |
| 悲观情景 | ¥XX.XX |
| 基准上行空间 | +XX% |
| 核心假设 | FY20XXE 营收 XXX 亿元，净利润 XXX 亿元，XX 倍 PE / XX% WACC |
| 宏观基准路径 | [实际利率回落、库存去化结束、美元走弱] |

**估值逻辑：** [1–2 句话]
**最大上行催化：** [一句话]
**最大下行风险：** [一句话]
**关键跟踪指标：** [2–3 个]
```

### 第二部分：分析报告

按以下板块组织，每块以叙述段落而非步骤清单呈现：

1. **公司概况与口径说明**——估值对象、并表范围、主要暴露、是否启用宏观模块及原因
2. **经营事实与历史归因**——近几年经营表现、增长来源、利润率驱动、现金流质量、执行与外部环境各自的贡献
3. **标准化盈利**——剥离一次性与宏观扰动后的真实盈利能力区间
4. **商业模式与竞争格局**——价值链位置、竞争优势、管理层与治理；若启用宏观模块，含宏观传导机制
5. **核心变量与预测**——1–3 个核心变量的选择依据、分业务/分驱动预测、利润率变化桥、假设来源标注
6. **宏观环境与情景路径**（仅启用时）——当前 regime、Bear/Base/Bull 路径、公司层映射
7. **估值**——方法选择、参数确定、EV 到每股价值桥接、资本回报策略
8. **情景与敏感性**——三情景目标价及驱动逻辑、最脆弱假设的敏感性测试
9. **反向校验与风险**——隐含预期拆解、与一致预期对比、反证处理
10. **结论与跟踪框架**——核心判断收敛、证伪条件、监控指标

**叙述原则**：关键判断的前提、推理链与反证条件须在叙述中自然呈现，读者应能看到"从 X 如何推到 Y、哪一跳最脆弱、什么会让它失效"，而非只看到结论。Wave 2/3 触发过的回退须在相应板块显式记录。

板块详略可按公司情况调整，某些板块可合并或简化。

## 防错原则

1. 订单额、定点金额、意向协议不等于收入；一次性收益不代表长期盈利能力；管理层指引是假设来源，不是事实。
2. 始终使用完全摊薄股本，不可忽略未来稀释。
3. 高增长不自动等同于高质量增长；行业叙事不能替代收入、利润率、现金流、股本这四个硬变量。
4. 经营模型未完成之前不可锚定估值结论；当前股价仅用于反向校验，不作为估值输入。
5. 景气高点利润上不叠加高倍数；商业模式不同的公司不可机械套用同业平均倍数。
6. 不将同一宏观利好同时计入销量、利润率和估值倍数。
7. 不将零利率或单一政策红利时期的估值中枢机械搬至当前 regime；regime change 出现时，历史均值与历史倍数的参考权重须主动下调。
8. 地缘政治须落到具体传导路径（需求 / 供给 / 成本 / 汇率 / 风险溢价），不能只是空泛风险条目。
9. 概率加权目标价中的概率是工作假设，不得伪精确到小数点。
10. 不跳过 Wave 2 定向深抓直接进入预测——Wave 1 的表层信息不足以支撑核心变量假设。
11. 第 4 步变量选择不是终局：Wave 2/3 的新发现必须触发显式回退，且回退过程要在报告中可见，不能被隐藏或事后合理化。
12. 不以叙述替代推演——关键判断须有可审计的推理链与反证条件；"因此""综合来看""考虑到"等叙述出现在关键节点视为推演缺失。
13. Wave 3 的目标是主动寻找推翻 base case 的证据，而非补强结论；找不到任何反证本身是警示信号，须立即补做一轮。
